# Small Farm Planner Tester Guide

## What This Is

Small Farm Planner is a game-like garden planning tool for backyard growers. It helps users place crops, understand companion planting rules, handle gardening tasks, record harvests, clean up beds, and plan the next season.

## Test Goal

Finish the 3-Min Check and decide whether the tool feels useful for planning a real home garden.

## What To Do

1. Open the planner.
2. Click "开始 3 分钟体验".
3. Follow the 3-Min Check from top to bottom.
4. Stop when the app says the experience path is complete.

## Feedback We Need

- Did you understand what the app is for within 30 seconds?
- Did each step make it clear where to click next?
- Did the companion/conflict explanation feel trustworthy?
- Did task, harvest, cleanup, and next-season planning feel connected?
- Where did you feel lost, confused, or overloaded?
- Would you use this again for a real backyard garden?

## Success Criteria

The first tester should be able to complete the core path in about 3 minutes without guidance from the maker.
